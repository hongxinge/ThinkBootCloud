# ThinkBootCloud Templates

提供开箱即用的部署模板和配置示例，开发者可按需复制到项目中使用。

## 模板列表

### 1. Docker Compose 部署模板
- `docker/docker-compose.yml` - 包含 Nacos、MySQL、Redis、RabbitMQ 完整环境
- `docker/.env.example` - 环境变量配置示例

### 2. CI/CD 模板
- `ci-cd/.github-workflows.yml` - GitHub Actions 构建和部署流程
- `ci-cd/gitlab-ci.yml` - GitLab CI/CD 配置

### 3. Maven Profile 模板
- `profiles/pom-profiles.xml` - 多环境 Maven Profile 配置

### 4. Nginx 配置模板
- `nginx/gateway.conf` - 网关反向代理配置
